A professional-grade **Retrieval-Augmented Generation (RAG)** chatbot developed during my GenAI Internship at **Capgemini**. This tool enables employees to interact with internal company policies, strategic initiatives, and core values using natural language, ensuring all responses are grounded in verified corporate data.

---

## 🌟 Key Features
- **Semantic Search:** Utilizes Google's `text-embedding-004` to understand the mathematical meaning behind queries, providing contextually relevant answers rather than simple keyword matching.
- **Source Citations:** Every response includes a "View Internal Sources" section, showing the exact PDF snippet and metadata used, ensuring transparency and trust.
- **Hallucination Mitigation:** By anchoring the LLM to a private vector database (ChromaDB), the system avoids the "hallucination" common in standard AI models.
- **Enterprise UI:** A clean, modern chat interface built with Streamlit, optimized for internal corporate use.

---

## 🏗️ Project Architecture

The system implements a modular RAG pipeline to ensure data privacy and accuracy:



1. **Ingestion:** Raw PDFs are loaded and cleaned using `PyPDFLoader`.
2. **Chunking:** Text is split using `RecursiveCharacterTextSplitter` into 1000-character segments with a 150-character overlap to preserve context.
3. **Embedding:** Chunks are transformed into vectors via the Gemini Embedding API.
4. **Storage:** Vectors are persisted in a local **ChromaDB** instance for lightning-fast retrieval.
5. **Retrieval:** The system fetches the top 3 most relevant context chunks based on the user's query.
6. **Generation:** **Gemini 2.5 Flash-Lite** synthesizes the final answer using only the retrieved facts.

🚀 Getting Started
1. Prerequisites
Python 3.10 or higher

A Google AI Studio (Gemini) API Key

2. Installation
Clone the repository and set up your environment:

Bash
git clone git@github.com:waseans/Chatbot_Capgemini.git
cd Chatbot_Capgemini
python -m venv venv
source venv/bin/activate  # On Windows: .\venv\Scripts\activate
pip install -r requirements.txt
3. Configuration
Create a .env file in the root directory and add your key:

Plaintext
GOOGLE_API_KEY=your_gemini_api_key_here
4. Running the Project
Phase 1: Build the Knowledge Base

Bash
python src/ingestion.py
Phase 2: Launch the Assistant

Bash
streamlit run src/app.py
